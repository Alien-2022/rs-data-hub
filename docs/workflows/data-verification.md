# 数据验证流程

## 🎯 目标

验证遥感数据集的论文链接和下载链接是否有效，确保数据集信息完整可靠。

---

## 📋 验证优先级

### 1. 论文链接验证

**优先级顺序**:
```
arxiv > IEEE/期刊官网 > GitHub/GitLab > 个人项目页面
```

| 平台 | 优先级 | 说明 |
|------|--------|------|
| **arxiv** | ⭐⭐⭐⭐⭐ | 开源预印本，访问稳定，优先查找 |
| **IEEE** | ⭐⭐⭐⭐ | 正式发表，但可能有付费墙 |
| **期刊官网** | ⭐⭐⭐⭐ | 如遥感学报、ISPRS 等，作为引导页面足够 |
| **GitHub/GitLab** | ⭐⭐⭐ | 项目代码仓库，可能包含论文 PDF |
| **个人项目页面** | ⭐⭐ | 可能失效，需定期验证 |

**查找策略**:
1. 先搜索 `数据集名称 + arxiv`
2. 找不到 arxiv 再搜期刊官网
3. 付费墙论文优先找 arxiv 预印本

---

### 2. 下载链接验证

**优先级顺序**:
```
官方平台 > 知名托管平台 > 大学官网 > 个人页面
```

| 平台类型 | 示例 | 可靠性 |
|---------|------|--------|
| **官方平台** | ISPRS、NODA 数据平台 | ⭐⭐⭐⭐⭐ |
| **知名托管** | Kaggle、GitHub、HuggingFace | ⭐⭐⭐⭐ |
| **大学官网** | 武大 GPCV、西工大 | ⭐⭐⭐ |
| **个人页面** | 个人 GitHub Pages | ⭐⭐ |

**验证要点**:
- [ ] 链接可访问（HTTP 200）
- [ ] 页面包含数据集描述
- [ ] 有明确的下载方式或申请流程
- [ ] 最后更新时间（确认是否维护）

## ✅ 验证检查清单

### 论文链接
- [ ] 链接可访问（HTTP 200）
- [ ] 页面包含论文标题和作者
- [ ] 与数据集相关
- [ ] 优先 arxiv 链接（开源）
- [ ] 记录 DOI（如有）

### 下载链接
- [ ] 链接可访问
- [ ] 页面包含数据集描述
- [ ] 有下载方式或申请流程
- [ ] 确认数据规模（图像数量、存储大小）
- [ ] 确认许可证类型

### 记录验证信息
- [ ] 验证时间
- [ ] 验证人
- [ ] 验证状态（✅ 有效 / ⚠️ 需申请 / ❌ 失效）
- [ ] 备注（如遇问题）

---

## 📊 验证状态分类

| 状态 | 符号 | 说明 | 处理方式 |
|------|------|------|---------|
| **有效** | ✅ | 链接正常，信息完整 | 直接记录 |
| **需申请** | ⚠️ | 链接正常但需邮件申请 | 记录申请流程 |
| **付费墙** | 🔒 | 论文需机构访问 | 找 arxiv 替代 |
| **失效** | ❌ | 链接 404 或无法访问 | 找替代链接 |
| **待验证** | ⏳ | 尚未验证 | 安排验证 |

---

## 🔄 验证流程

```
1. 收集数据集名称
         ↓
2. 搜索论文链接（arxiv 优先）
         ↓
3. 验证论文链接（浏览器/命令行）
         ↓
4. 搜索下载链接（官方平台优先）
         ↓
5. 验证下载链接
         ↓
6. 记录验证结果（状态 + 时间 + 验证人）
         ↓
7. 更新数据集清单
```

## 💡 经验技巧

### 快速查找 arxiv 论文
```
搜索格式：[数据集名称] + arxiv
示例：DIOR remote sensing arxiv
```

### 处理付费墙
- IEEE 论文 → 搜 arxiv 预印本
- ScienceDirect → 搜 arxiv 或 ResearchGate
- Springer → 搜 arxiv 或作者个人页面

### 找替代下载源
- 官方页面失效 → 搜 Kaggle/GitHub
- GitHub 404 → 搜作者新仓库或联系作者
- 大学官网需申请 → 邮件联系作者
