## 🔧 Supabase 配置

### RLS 策略配置 (重要 ⭐⭐⭐⭐⭐)
**优先级**: 高  
**原因**: 保护数据安全，允许公开读取

```sql
-- 1. 启用 RLS
ALTER TABLE datasets ENABLE ROW LEVEL SECURITY;

-- 2. 创建公开读取策略
CREATE POLICY "Public read access" ON datasets
  FOR SELECT
  USING (true);

-- 3. 验证策略
SELECT * FROM pg_policies WHERE tablename = 'datasets';
```

**操作步骤**:
1. 打开 Supabase Dashboard → SQL Editor
2. 执行上述 SQL
3. 验证策略已创建
