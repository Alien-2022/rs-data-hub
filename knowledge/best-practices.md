## 📊 数据收集最佳实践

### ✅ 优先选择有论文的数据集

**原则**: 有参考论文的数据集更可靠

**理由**:
- 经过学术审查
- 元数据更完整
- 引用信息齐全
- 更容易找到官方下载源

**执行标准**:
```
✅ 优先：有 arxiv 论文 + 官方下载链接
✅ 接受：有期刊论文 + 官方下载链接
⚠️ 谨慎：无论文但有知名平台托管
❌ 避免：无论文 + 个人页面托管
```

---

### ✅ 多维度元数据收集

**原则**: 字段完整性 > 字段齐全度

**必填字段** (核心，不能为空):
- name (名称)
- description (描述)
- task_types (任务类型)
- data_modality (数据模态)
- image_count (图像数量)
- storage_size (存储大小)
- publisher (发布机构)
- publish_year (发布年份)
- download_url (下载链接)

**可选字段** (有就填):
- paper_url (论文链接)
- project_url (项目首页)
- doi (DOI)
- sensor_name (传感器名称)
- spatial_resolution (空间分辨率)
- spectral_bands (波段数量)

**执行标准**:
```
✅ 合格：必填字段 100% 完整
✅ 优秀：必填字段 100% + 可选字段 50%+
❌ 不合格：必填字段有缺失
```

## 🔍 数据验证最佳实践

### ✅ arxiv 优先策略

**原则**: 优先使用 arxiv 预印本作为论文链接

**理由**:
- 开源免费，无付费墙
- 访问稳定，不易失效
- 可直接下载 PDF
- 有标准引用格式

**执行流程**:
```
1. 搜索 [数据集名称] + arxiv
2. 验证 arxiv 链接可访问
3. 记录 arxiv ID 和 DOI
4. 如无 arxiv，再找期刊官网
```

**成功率**: 遥感领域 ~80%+ 有 arxiv 预印本

---

### ✅ 多平台验证策略

**原则**: 重要数据集找多个下载源

**理由**:
- 单平台可能失效
- 多平台提高可用性
- 不同平台可能有不同版本

**执行标准**:
```
✅ 优先：官方平台 (ISPRS、NODA 等)
✅ 接受：知名托管 (Kaggle、GitHub、HuggingFace)
⚠️ 备选：大学官网、个人页面
```

**记录要求**:
- 主下载源（优先推荐）
- 备用下载源（主源失效时使用）

---

### ✅ 验证即记录

**原则**: 验证过的链接立即记录状态

**记录内容**:
- 验证时间
- 验证人
- 验证状态（✅/⚠️/❌）
- 验证方法（浏览器/curl/web_fetch）
- 备注（如遇问题）

**记录模板**:
```markdown
**论文链接**:
- 链接：https://arxiv.org/abs/1909.00133
- 状态：✅ 有效
- 验证时间：2026-03-27 01:00
- 验证人：小龙虾
- 验证方法：浏览器

**下载链接**:
- 链接：https://www.kaggle.com/datasets/xxx
- 状态：✅ 有效
- 验证时间：2026-03-27 01:05
- 验证人：小龙虾
- 验证方法：浏览器
```

## 🛠️ 工具使用最佳实践

### ✅ 浏览器验证为主，工具为辅

**原则**: 重要链接必须手动验证

**工具选择**:
```
✅ 浏览器验证 - 最可靠，适合少量验证
✅ curl 命令 - 快速检查 HTTP 状态码
⚠️ web_fetch - 辅助工具，可能受反爬影响
❌ 仅依赖自动化 - 容易误判
```

**验证流程**:
```
1. 浏览器手动验证（核心链接）
2. curl 快速检查（批量验证）
3. web_fetch 提取内容（辅助确认）
```

## 📈 质量保障最佳实践

### ✅ 100% 验证目标

**原则**: 收录的数据集必须 100% 验证链接有效

**执行标准**:
```
✅ 收录标准：论文链接 + 下载链接均验证有效
⚠️ 暂缓收录：任一链接失效或需申请
❌ 不收录：两个链接均失效
```

**实际成果**:
- 2026-03-27: 20/20 (100%) 验证成功

---

### ✅ 定期复查

**原则**: 已验证链接定期复查

**复查周期**:
```
核心链接（arxiv、Kaggle） → 每 6 个月
重要链接（期刊官网） → 每 3 个月
个人页面 → 每 1 个月
```

**复查内容**:
- 链接是否可访问
- 页面内容是否变更
- 数据集是否更新版本
