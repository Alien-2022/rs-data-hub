# 知识库导航

## 📚 知识库说明

这里存放的是**经验性知识**——在项目中积累的验证技巧、踩坑记录、最佳实践等。

与 `docs/` 的项目文档不同，这里的文档是**可进化的**，会随着项目经验持续更新。

---

## 🗂️ 知识库文件

### 1. [验证经验](./verification-experience.md)
**用途**: 数据集链接验证的技巧、工具、策略

**包含内容**:
- 论文链接验证优先级（arxiv > IEEE > 期刊官网）
- 下载链接验证策略（多平台确认）
- 常用工具与命令
- 验证检查清单

**何时查阅**: 需要验证新数据集链接时

---

### 2. [踩坑记录](./lessons-learned.md)
**用途**: 记录项目中遇到的问题和解决方案

**包含内容**:
- 技术问题（数据库、API、部署等）
- 流程问题（验证失败、链接失效等）
- 解决方案与预防措施

**何时查阅**: 遇到类似问题时快速查找解决方案

---

### 3. [最佳实践](./best-practices.md)
**用途**: 工作流程优化建议

**包含内容**:
- 数据收集最佳实践
- 验证工作最佳实践
- 协作效率提升技巧
- 工具使用建议

**何时查阅**: 优化工作流程时参考

---

## 🔄 知识更新流程

```
每日任务 (memory/)
    ↓
发现新经验/踩坑
    ↓
整理到知识库 (knowledge/)
    ↓
更新相关工作流程 (docs/workflows/)
```

---

## 📝 使用建议

1. **任务前** - 查阅相关工作流程和最佳实践
2. **任务中** - 遇到问题查阅踩坑记录
3. **任务后** - 将新经验总结到知识库
